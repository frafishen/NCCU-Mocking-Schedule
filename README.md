# NCCU 選課小精靈(網頁版)

網頁本體: [https://frafishen.github.io/NCCU-Mocking-Schedule/](https://frafishen.github.io/NCCU-Mocking-Schedule/)

政大的選課系統並沒有模擬課表的功能，所以大家都要自己拉excel表格。
